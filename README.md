# Описание программы Simple Test
  Программа Simple Test создана для создания тестов для последующего тестирования в образовательных учреждениях и дома. Программа снабжена инструментом для отслеживания выполненных тестов на локальном компьютере.  
  Как и любая другая программа, Simple Test имеет доступ к основным своим функциям из главного меню.  

## Главное меню
Главное меню Simple Test выглядит следующим образом:  
![главное меню Simple Test](http://cs7003.vk.me/c614823/v614823236/a838/e-fKKQ3-tjY.jpg)

Все довольно просто. Три кнопки для вызова главных плагинов программы:

* Начать тестирование. Клик по кнопке вызывает плагин тестирования

* Создать/Редактировать тест. Клик по кнопке вызывает плагин для создания и редактирования тестов.

* Просмотреть результаты. Клик по кнопке открывает сохраненные результаты выполненных тестов на данном компьютере.

# Основные элементы окна
Окно создания теста разделен на несколько логических панелей:

+ Параметры теста
+ Панель изменения/создания вопроса
+ Редактор вариантов ответа
+ Панель управления вопросами.

## Панель параметров теста
На панели параметров теста расположены поля для редактирования параметров теста. В текущей версии программы доступен только параметр названия теста и кнопки сохранения теста и загрузки ранее созданного теста.  
Параметр "Название теста". Будет отображаться в заголовках теста, а так же списках выполненных тестов для их идентификации.
    
## Редактирование вопроса
![Панель редактирования вопроса](http://cs7003.vk.me/c617330/v617330236/4901/41hZ8gulsE0.jpg)

В данной панели можно добавлять новые вопросы, а так же редактировать существующие вопросы теста.  
## Добавление вопроса
Для того, чтобы добавить вопрос к текущему тесту, необходимо заполнить два поля:  

* Текст вопроса
* Тип вопроса
    
После этого нажать кнопку добавить. После этого вопрос появиться в списке вопросов на панели управления вопросами.
Поле "Тип вопроса", предлагает на выбор 3 типа вопроса:

* Один правильный ответ. (Из всех вариантов ответа - один верный)
* Несколько правильных ответов. (Несколько вариантов ответа могут быть верными)
* Разложить в правильном порядке. (Все варианты ответа веры, но их необходимо расположить в правильном порядке)

## Панель управления вопросами
![Панель управления вопросами Simple Test](http://cs7003.vk.me/c617330/v617330236/4908/kAJWPwCsiuk.jpg)

Данная панель отображает все вопросы текущего теста, а также кнопки для управления ими:  
    
* Редактировать. Кнопка активирует панель редактирования вопроса, где Вы сможете изменить текст и тип вопроса. Чтобы сохранить изменения, необходимо нажать кнопку "Применить изменения". Если Вы по ошибке нажали редактирование вопроса, можно отменить действие, нажав кнопку "отмена" в панели редактирования вопроса.      
* Редактирование вариантов ответа. Кнопка активирует панель редактора вариантов ответа.     
* Удалить. Кнопка удаляет выделенный вопрос из теста.

## Редактор вариантов ответа
![Редактор вариантов ответа](http://cs7003.vk.me/c617330/v617330236/490f/Dt0_FM3jk9Y.jpg)

Данная панель предназначена для добавления/изменения/удаления вариантов ответа у вопроса.

* Добавить. Чтобы добавит вариант ответа к вопросу, необходимо заполнить поле в панели "добавить новый вариант ответа" и нажать кнопку "добавить". После этого новый вариант ответа сразу появиться в списке вариантов ответа текущего вопроса. Внимание. При добавлении вариантов ответа для вопроса типа "расположить в правильном порядке", необходимо добавлять сразу в правильном порядке. Во время тестирования варианты ответа случайно меняются местами.    
* Редактировать. Чтобы редактировать вариант ответа, необходимо его выделить в списке вариантов ответа, затем изменить текст варианта ответа в соответствующем поле и нажать "Сохранить". По умолчанию, всем вариантам ответа присваивается значение "Не верно", поэтому после того, как измените текст варианта ответа, поставьте галочку на опции "Правильный ответ", всё в той же панели.  
В зависимости от типа вопроса, количество верных ответов может быть разное:

* Один ответ = 1
* Несколько верных ответов >= 1
* Расположить в правильном порядке == Все ответы верные.
  
* Удалить. Чтобы удалить вариант ответа, необходимо выделить вариант ответа и нажать кнопку удалить.

После завершения всех действий с вариантами ответа необходимо нажать кнопку "Ок", чтобы сохранить изменения.

## Сохранение теста  
После того, как вы тест готов, Вы можете его сохранить, чтобы пользоваться Вашим трудом, как обычным файлом на компьютере:

* Переносить на другие устройства
* Редактировать если во время тестирования были найдены ошибки
* Хранить
* Использовать

Для того, что сохранить тест, необходимо обязательно ввести название теста и нажать кнопку "Сохранить".  
После сохранения, создается файл с указанным именем.
Внимание. В файле находятся структурированные, не бинарные данные. Поэтому, если вы хотите, чтобы никто не знал заранее вопросов и ответов к ним, нужно припрятать данный файл.

## Загрузка теста
Ранее созданный тест также можно редактировать и добавлять новые вопросы. Для этого можно воспользоваться кнопкой "Загрузить тест".  
Тест загрузится и все необходимые данные сразу появятся в соответствующих полях.

# Тестирование
Чтобы начать тестирование, необходимо в главном меню нажать на кнопку "Начать тестирование". Откроется окно тестирования.

![Окно тестирования в Simple Test](http://cs7003.vk.me/c617330/v617330236/4921/MHd2NFKB3Eg.jpg)

## Шаг 1. Подготовка к тестированию.
Прежде чем приступить к тестированию, необходимо ввести все данные:

* Загрузить тест
* Имя
* Фамилия
* Группа

После нажатия на кнопку загрузить, программа попросит Вас ввести пароль. Это так же сделано дял предотвращения жульничества. Пароль можно узнать у меня.

После того, как все данные были введены, нужно нажать на кнопку "Начать тест".

## Шаг 2. Тестирование
Тестирование заключается в ответе на вопросы следующего вида:
* Один правильный ответ
![Один правильный ответ](http://cs7003.vk.me/c617330/v617330236/4933/8AHGUp9-1HY.jpg)

* Несколько правильных ответов
![Несколько правильных ответов](http://cs7003.vk.me/c617330/v617330236/493c/qgODfYimCJE.jpg)

* Расставить в правильном порядке
![Разложить в правильном порядке](http://cs7003.vk.me/c617330/v617330236/4945/WRBOK349ZgM.jpg)

После прохождения теста, программа выводит результат в виде:

<количество правильных ответов>/<всего вопросов> <процент правильных ответов>%

![Резльутат](http://cs7003.vk.me/c617330/v617330236/492a/chX0s8fatwU.jpg)

#Отслеживание выполненных тестов
В главном меню расположена кнопка "Просмотреть результаты". После нажатия её и ввода пароля, появляется лог выполненных  тестов на данном компьютере:

![Результат тестов на данном компьютере](http://cs619818.vk.me/v619818236/91f/OFhoiiC581U.jpg)

* Дате выполнения

* Названии теста.
